Frameworks

Definition

There are basically 2 types of frameworks: backend and frontend, depending on whether the framework is for the presentation layer or the application/ logical layer.

In the world of web design, a framework is defined as a package made up of a structure of files and folders of standardized code (HTML, CSS, JS documents etc.) which can be used to support the design and evelopment of websites or web applications. In this way, frameworks allow us to save time.

The usual components of a frontend framework are:

  • A page layout grid, which allows to position the different elements that make up the site design in a simple, versatile and responsive structure.
  • Typography style definitions for HTML elements.
  • Cross browser compatibility to ensure consistent displays across all browsers.
  • Standard CSS classes which can be used to style simple and advanced components of the user interface.
  • Responsiveness features to ensure the flexibility of the display in most devices.
  • Accessibility for assistive technology users.

Framework products

There are several frameworks that are free and easy to adopt.

A few of them are robust and maintained well enough to be more reliable.

Bootstrap was chosen for the Embedded Ethics website. This framework is widely used across Harvard for both web sites and web applications.

As Bootstrap evolves and provides more stability and spphisticated features, it can easily be upgraded and the design can be easily scaled to the new features and functionality.

Bootstrap

  • Bootstrap is a free front-end framework for faster and easier web development
  • Bootstrap includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and many other, as well as optional JavaScript plugins
  • Bootstrap also gives you the ability to easily create responsive designs

Why use Bootstrap?

  • Easy to use: Anybody with just basic knowledge of HTML and CSS can start using Bootstrap
  • Responsive features: Bootstrap's responsive CSS adjusts to phones, tablets, and desktops
  • Mobile-first approach: In Bootstrap, mobile-first styles are part of the core framework
  • Browser compatibility: Bootstrap 4 is compatible with all modern browsers (Chrome, Firefox, Internet Explorer 10+, Edge, Safari, and Opera)

Bootstrap grid system

Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.

If you do not want to use all 12 columns individually, you can group the columns together to create wider columns:

span 4 span 8
span 6 span 6
span 12

The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

Make sure that the sum adds up to 12 or fewer (it is not required that you use all 12 available columns).
